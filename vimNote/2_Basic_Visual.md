# 可视

### 可视模式

`v` # 字符

`V` # 行

`<C-v>` # 块

 

`gv` # 重选上次高亮 

 

退出：

`<Esc>/<C-c>`

`<C-N>`

 

查看帮助：

`v_`

`v_b_`



### 与其他指令联合使用

删除：

`d`

`D`  




复制：

`y`
`Y`

 

修改：

`c` # 仅当少于等于一行时生效

`C` # 选中至行尾

 

合并：

`J`

`gJ` # 不加入空格

 

### 缩进

`>`
`<`



### 查看帮助

`K` # 使用man执行选中
