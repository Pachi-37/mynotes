# 操作命令

基于单词的移动

` w ` # 跳转到下一个单词开头

` e ` # 跳转到下一个单词结尾

` b ` # 跳转到上一个单词结尾

 

基于一行的移动

` $ ` # 跳转到行尾

` 0 ` # 移动到第一个非空的字符

` ^ ` # 移动到第一个非空的字符

 

文本移动

'`G ` # 移动到文末

` gg ` # 移动到文首 



### 行中快速搜索

` f ` # 查询f后面指定的字符 

该命令添加数字指定 

 \# 例： 5f`<space>` 

` F `  # 反向查询



### 删除 & 更改

` dd ` # 删除一行

` dw ` # 删除后一个单词

 

` c ` # 改变文本——（删除并插入）

` r ` # 替换一个字符

` ~ ` # 改变字符的大小写.如果指定了计数，则会更改计数字符。

 

` J ` # 将当前行与下一行连接起来。在第一行的末尾添加一个空格



使用删除`d`可以与移动指令配合使用标记命令



### 重复上一步

` . ` # 重复上一步对文本更改的操作



### 宏指令

` q【letter】 ` # 开启宏录制

` @【letter】 ` #  执行宏指令
